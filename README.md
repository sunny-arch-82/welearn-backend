# 🎓 WeLearn: AI-Assisted Course Planning Platform

An AI-powered course planning platform that converts minimal user input into **structured learning paths with curated educational resources**.

WeLearn combines **AI-generated syllabus creation with intelligent multi-source retrieval** to generate complete learning roadmaps for any topic.

> This project demonstrates how AI systems can power **automated curriculum design and educational discovery platforms**.

---

## ✨ Features

### 🧠 AI-Generated Learning Paths
WeLearn transforms simple user inputs into structured learning plans.

Users provide:

- Topic or skill to learn
- Learning duration
- Optional learning goals

The system generates:

- Structured course modules
- Weekly learning roadmap
- Topic breakdowns
- Curated educational resources

---

### 🔎 Multi-Source Educational Retrieval

WeLearn automatically gathers high-quality learning resources from trusted sources.

Integrated sources include:

- Google Custom Search
- YouTube educational content
- GitHub repositories
- Developer documentation

Domain filtering prioritizes trusted learning platforms such as:

- Coursera
- edX
- GitHub
- official documentation

---

### ⚡ Intelligent Query Processing

The system improves retrieval quality using:

- Query normalization
- Domain filtering
- Resource ranking
- Fallback retrieval strategies

These techniques significantly improve **resource relevance and content quality**.

---

### 🧩 Modular Backend Architecture

The backend is implemented as a modular Python service.

Core components manage:

- request parsing
- AI syllabus generation
- resource retrieval
- response aggregation

This architecture allows seamless integration with **web or mobile frontends**.
